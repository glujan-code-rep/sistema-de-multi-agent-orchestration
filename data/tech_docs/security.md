# Acceso y Seguridad IT - Guía

## Credenciales y Acceso

### Creación de cuenta
1. HR envía datos del nuevo empleado a TI
2. TI crea cuenta en 24 horas
3. Usuario recibe credenciales por email seguro
4. Primer login debe cambiar contraseña

### Recuperación de contraseña
- Opción 1: Self-service en portal de login
- Opción 2: Llamada a IT Support
- Opción 3: Visitando mostrador de TI
- Requiere validación de identidad

### Bloqueo de cuenta
- Después de 5 intentos fallidos
- Desbloqueo automático después de 30 minutos
- Para desbloqueo inmediato: contactar TI

## Acceso Remoto

### VPN Corporativa
- Disponible para todos los empleados
- Requiere autenticación de dos factores
- Solo acceso a sistemas internos
- No usar para downloads grandes

### Escritorio Remoto
- Solo para personal autorizado
- Solicitar acceso a sistemas específicos
- Conexión segura obligatoria

## Políticas de Seguridad

### Uso aceptable
- Solo fines laborales
- No instalar software no autorizado
- No compartir credenciales
- No almacenar datos sensibles localmente

### Dispositivos
- Equipos corporativos son propiedad de empresa
- Solo uso permitido personal mínimo
- Reporting de pérdida inmediata
- Devolución al causar baja

### Datos
- Clasificación: Público, Interno, Confidencial
- Confidencial requiere encryption
- No enviar por email sin加密
- Shredding de documentos físicos

## Reporte de Incidentes

### Qué reportar
- Phishing o intentos de fraude
- Pérdida de equipo
- Acceso no autorizado
- Malware o virus detectado
- Comportamiento sospechoso
