# Guía de Conectividad y Redes - IT

## Problemas de WiFi

### No puedo conectar
1. Verificar que WiFi esté habilitado (tecla o configuración)
2. Olvidar la red y reconectar
3. Verificar contraseña correcta
4. Reiniciar el equipo
5. Apagar y prender el router

### Conexión lenta
- Acercarse al router
- Fewer dispositivos conectados
- Cambiar de banda (2.4GHz vs 5GHz)
- Actualizar firmware del router
- Considerar usar cable Ethernet

## VPN Corporativa

### Configuración inicial
1. Solicitar credenciales a TI
2. Descargar cliente VPN del sitio interno
3. Instalar con permisos de administrador
4. Conectar usando credenciales corporativas

### Problemas comunes
- "No se puede establecer conexión": Verificar internet
- "Credenciales incorrectas": Verificar mayúsculas/minúsculas
- "Certificado no válido": Actualizar cliente VPN

## Correo Electrónico

### Outlook no conecta
1. Verificar conexión a internet
2. Cerrar y abrir Outlook
3. Eliminar y agregar cuenta de nuevo
4. Verificar configuración de servidor
5. Contactar soporte

### Adjuntos no se abren
- Verificar tamaño (máx 25MB)
- Tipos permitidos: doc, pdf, xls, ppt, images
- Comprimir archivos muy grandes
- Verificar que no estén bloqueados

## Impresoras

### Impresora no responde
1. Verificar conexión (USB o red)
2. Verificar que tenga papel y tinta
3. Reiniciar impresoras y equipo
4. Reinstallar driver
5. Verificar cola de impresión

### Impresora en red no se encuentra
- IP debe estar en mismo segmento
- Verificar que esté encendida
- Ping a la IP de la impresora
- Reinstallar driver de red
