# Software y Aplicaciones Corporativas - IT

## Instalación de Software

### Software autorizado
Solo se puede instalar software de la lista approved:
- Microsoft Office 365 (incluido)
- Adobe Acrobat Reader
- Chrome, Firefox, Edge
- 7-Zip
- Teams, Slack, Zoom

### Solicitud de software no autorizado
1. Crear ticket en portal de TI
2. Especificar razón de negocio
3. Aprobación de gerente requerido
4. Evaluación de seguridad
5. Instalación por equipo de TI

## Actualizaciones

### Windows Update
- Se ejecutan automáticamente los martes
- Reinicio requerido dentro de 48 horas
- No posponer más de 3 veces
- Guardar trabajo antes de reiniciar

### Aplicaciones
- Microsoft Office: Actualizaciones mensuales
- Navegadores: Según disponibilidad
- Otros: Según cronograma de TI

## Acceso a Sistemas

### Sistemas internos
- Portal empleados: portal.empresa.com
- CRM: crm.empresa.com
- ERP: erp.empresa.com
- Wiki interno: wiki.empresa.com

### Credenciales
- Usuario: RFC o número de empleado
- Password: Se cambia cada 90 días
- Bloqueo: Después de 5 intentos fallidos
- Recuperación: A través de IT Support

## Seguridad

### Contraseñas seguras
- Mínimo 12 caracteres
- Mayúsculas, minúsculas, números, símbolos
- No usar información personal
- Diferente para cada sistema

### Phishing
- No hacer clic en links sospechosos
- Verificar remitente
- No proporcionar credenciales por email
- Reportar a seguridad@empresa.com

## Backups

### Datos personales (OneDrive)
- Sync automático activaddo
- Versioning de 30 días
- Restaurar archivos eliminados

### Datos corporativos
- No almacenar en disco local
- Usar siempre unidades compartidas
- Solicitar respaldo si es necesario
