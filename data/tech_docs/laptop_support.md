# Guía de Soporte Técnico - Laptop y Equipos

## Problemas de Lentitud

### Causas comunes
1. Muchos programas abiertos simultáneamente
2. Disco duro casi lleno
3. malware o virus
4. Actualizaciones pendientes
5. Memoria RAM insuficiente

### Soluciones paso a paso
1. Cierra aplicaciones que no uses
2. Verifica espacio en disco (mínimo 10% libre)
3. Ejecuta antivirus completo
4. Instala las últimas actualizaciones de Windows/macOS
5. Añade más RAM si es posible

### Cuándo escalar
- Si el problema persiste después de seguir los pasos
- Si escuchas ruidos extraños del disco
- Si el equipo se apaga solo

## Problemas de Batería

### Optimización de batería
- Reduce brillo de pantalla
- Cierra aplicaciones en background
- Desconecta dispositivos USB no necesarios
- Usa modo de ahorro de energía

### Batería que no carga
1. Verifica que el cargador esté bien conectado
2. Prueba otro enchufe
3. Limpia los puertos de carga
4. Prueba otro cargador (si está disponible)
5. Contacta a TI si persiste

## Pantalla y Video

### Pantalla en negro
- Ajusta brillo con teclas Fn + brillo
- Conecta a monitor externo para probar
- Reinicia el equipo
- Actualiza drivers de video

### Problemas de resolución
- Clic derecho en escritorio > Configuración de pantalla
- Selecciona resolución recomendada
- Actualiza drivers de tarjeta gráfica

## Conectividad

### WiFi no funciona
1. Verifica que WiFi esté habilitado
2. Olvídate de la red y reconéctate
3. Reinicia el router
4. Olvídate de todas las redes y reconecta
5. Actualiza drivers de WiFi

### Ethernet no conecta
- Verifica cable bien conectado
- Prueba otro puerto en el switch
- Configura IP manualmente si DHCP falla
- Verifica que el puerto no esté deshabilitado
